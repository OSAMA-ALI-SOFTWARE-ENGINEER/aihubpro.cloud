# AIHubPro Cloud

AIHubPro is a centralized platform for AI tools, tutorials, and real-world case studies helping professionals turn artificial intelligence into measurable outcomes.

## Development Setup

This project is built with React + Vite.

### Prerequisites
- Node.js (v18 or higher)
- npm

### getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

This project is configured to automatically deploy to Hostinger via GitHub Actions on push to the `main` branch.
